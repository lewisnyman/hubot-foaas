# hubot-foaas

Basic interface for FOAAS.com

See [`src/foaas.coffee`](src/foaas.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-foaas --save`

Then add **hubot-foaas** to your `external-scripts.json`:

```json
[
  "hubot-foaas"
]
```
